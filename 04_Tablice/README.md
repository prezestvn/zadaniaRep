## Zadanie 2

```JavaScript
Przykładowa tablica:  [4, 55, 17, 22, 1, 43, 53, 11, 9, 20];
Zmodyfikowana tablica: [4, 56, 18, 22, 2, 44, 54, 12, 10, 20];
```
